Purpose

This Stellar System Generator primarily aims to build a JSON-File describing the generated system. In a second step the generated description is visualized as a system diagram. The diagram is not to scale and its purpose is to give rather a technical overview of astronomical data (e.g. rotation axis, tidal lock, etc.) than a photographic render with textures.

If provided, the generator will populate a template-file with missing values. This way the output of the generator is highly customizable.
